# Adversarial-Chess-Solver

A Python-based chess game and solver that uses Artificial Intelligence algorithms to play chess. It includes a Graphical User Interface (GUI) for an interactive playing experience.

## Overview
This project implements adversarial search algorithms (like Minimax) to calculate the best moves in a chess game, allowing users to play against a smart AI opponent.
